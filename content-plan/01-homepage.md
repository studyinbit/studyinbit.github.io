# Homepage - Content Plan

Create an unforgettable first impression that immediately differentiates BIT from generic university websites. Should be bold, modern, and instantly communicate value to ambitious Indonesian students.

## Design Notes
- **Aesthetic Direction**: Modern tech-forward with warmth, light mode
- **Typography**: Clash Display (hero headlines), Plus Jakarta Sans (body)
- **Color Palette**:
  - Primary: Warm gradient (amber #F59E0B to orange #EA580C)
  - Secondary: Cool gradient (blue #3B82F6 to purple #8B5CF6)
  - Accent: Emerald #10B981 (for CTAs)
  - Background: White with subtle gradients
- **Navigation**: Pill-shaped blur nav bar (glassmorphism)
- **Animations**:
  - Hero: Staggered fade-in with slide-up (300ms delays)
  - Stats: Count-up animation on scroll into view
  - Cards: Hover lift with glow effect
  - Gradient overlays: Slow, organic movement
- **Layout**: Asymmetric hero, overlapping elements, generous spacing
- **Configurable**: Make content easily editable for future updates
